# 合歡東峰

## 基本資訊

- **海拔高度**：3,421 公尺（合歡群峰中第二高）
- **所屬山脈**：中央山脈
- **行政區位置**：南投縣仁愛鄉、花蓮縣秀林鄉交界
- **百岳排名**：百岳之一，合歡群峰知名度最高的山頭之一
- **難度等級**：A 級（新手級）
- **單程時間**：木棧道平緩好走，來回約 2～3 小時

合歡東峰登山口與奇萊山登山口相近，雖然是合歡群峰中第二高的山頭，但步道以木棧道為主，十分平緩，是條老少咸宜的路線。過去東峰東側曾設立滑雪場（已拆除），如今則因四季植被變化著稱：夏季有玉山杜鵑花海，冬季可見白雪覆蓋的冷杉林。

## 申請流程

- **入園證**：不需申請。合歡東峰不在太魯閣國家公園生態保護區範圍內。
- **入山證**：需申請。與合歡山主峰相同，須於入山前 5～30 天內向警政署線上申請入山證，登山當日隨身攜帶。
- 申請網址：內政部警政署入山案件申辦系統 / 臺灣登山申請一站式服務網（https://hike.taiwan.gov.tw/）。

## 路線與住宿

- **主要登山口**：松雪樓附近，與奇萊山登山口共用入口。
- **建議天數**：1 日往返即可，不需過夜。
- **沿途住宿**：建議安排 2 天 1 夜行程時，入住清境地區民宿或小木屋。
- 常見安排：與合歡主峰、石門山串成一日多峰行程，新手一次收集三座百岳。

## 安全須知

- **常見風險**：階梯較多（俗稱「階梯地獄」），雖然平緩但仍需注意體力分配；高山環境仍有高山症風險。
- **天氣多變**：山區午後容易起濃霧，影響視野與沿途風景拍攝，建議上午前往。
- **季節特色**：4～6 月杜鵑花季為最佳賞花期；冬季偶有降雪，是熱門賞雪地點之一，但需注意保暖與路面結冰。
- **建議裝備**：保暖外套、防風衣物、登山鞋；冬季前往應加強防寒裝備。
